# CSP SMB Security Enablement Guide

A practical, field-ready repository for Microsoft CSP resellers to:

- get ready to sell SMB security solutions,
- prepare offers and customer scenarios,
- and drive successful adoption and deployment.

This guide is aligned to the Microsoft partner motion and uses the SMB security growth hub as a reference point:

- [SMB Security Growth Hub](https://microsoftpartners.microsoft.com/Microsoft-Security-Partners/Grow-SMB-Security/)

## Who this is for

- CSP sales leaders
- solution sales specialists
- technical pre-sales and architects
- customer success and deployment teams

## What is in this repo

- Sales readiness 
- Tech readiness and deployment guides
- Learning materials and playbooks
- Available demos
