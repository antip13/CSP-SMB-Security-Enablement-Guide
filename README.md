# CSP SMB Security Enablement Guide

A practical, field-ready repository for Microsoft CSP resellers to:

- get ready to sell SMB security solutions,
- prepare offers and customer scenarios,
- and drive successful adoption and deployment.

This guide is aligned to the Microsoft partner motion and uses the SMB security growth hub as a reference point:

- https://microsoftpartners.microsoft.com/Microsoft-Security-Partners/Grow-SMB-Security/

## Who this is for

- CSP sales leaders
- solution sales specialists
- technical pre-sales and architects
- customer success and deployment teams

## What is in this repo

- `docs/00-overview.md`: Program goals, outcomes, and operating model
- `docs/01-sales-readiness.md`: Sales readiness framework and execution checklist
- `docs/02-offer-packaging.md`: Offer design for Defender/Purview + Business Premium scenarios
- `docs/03-scenario-playbook.md`: Repeatable scenario motions by SMB profile
- `docs/04-adoption-deployment.md`: Deployment and adoption approach for customer success
- `templates/customer-discovery.md`: Discovery script and qualification worksheet
- `templates/security-assessment-checklist.md`: Pre-sales security baseline checklist
- `templates/90-day-adoption-plan.md`: Post-sale adoption planning template

## Quick start

1. Read `docs/00-overview.md` to align your team and define ownership.
2. Run the readiness checklist in `docs/01-sales-readiness.md`.
3. Select and package one offer from `docs/02-offer-packaging.md`.
4. Use `docs/03-scenario-playbook.md` to execute one customer motion.
5. Deliver with `docs/04-adoption-deployment.md` and use the templates to support adoption planning and customer execution.
